# TwoSigma — Платформа для A/B тестирования

## Статус
📅 Последнее обновление: 13 марта 2026 г.
🔄 Статус: ✅ Рабочий прототип

## Описание
Аналог платформы **TriSigma от Авито** для управления A/B тестами и экспериментов.
Покрывает полный цикл A/B-тестирования: от заведения экспериментов до аналитики.

## Целевой функционал (по аналогии с TriSigma)

### 🧪 Управление экспериментами
- [ ] Создание/редактирование/удаление A/B тестов
- [ ] Feature Flags (фича-флаги) для управления функциями
- [ ] Сплитование пользователей (детерминированный хэш-алгоритм)
- [ ] Поддержка множественных экспериментов одновременно
- [ ] Визуальный редактор тестов (без кода)

## Архитектурные решения
- **Backend**: Python + Streamlit
- **Базы данных**: SQLAlchemy (поддержка PostgreSQL, MySQL, SQLite)
- **Статистика**: SciPy, Statsmodels
- **Визуализация**: Plotly, Altair
- **Хэширование**: hashlib (детерминированный сплит)
