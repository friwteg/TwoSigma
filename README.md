# TwoSigma — Платформа для A/B тестирования

TwoSigma — аналитическая платформа для A/B-тестирования, позволяющая управлять экспериментами, собирать и анализировать метрики, а также оценивать статистическую значимость результатов. Проект демонстрирует навыки работы с данными, статистического анализа, SQL, визуализации и построения аналитических инструментов.  
Ссылка на платформу — https://twosigma.streamlit.app/

## 📋 Описание

TwoSigma покрывает полный цикл A/B-тестирования:
- Создание и управление экспериментами
- Feature Flags для управления функциями
- Детерминированное сплитование пользователей
- Расчёт метрик и статистической значимости
- Автоматическая подгрузка данных из подключённых БД
- Интерактивные дашборды и визуализация


## 🛠️ Технологии

- **Python 3.10+**
- **Streamlit** — UI фреймворк
- **SQLAlchemy** — ORM для работы с БД
- **Pandas** — обработка данных
- **SciPy** — статистический анализ
- **Plotly** — интерактивная визуализация
- **Pydantic** — валидация данных

P.S. Вдохновлено платформой для a/b тестов TriSigma от Авито.
